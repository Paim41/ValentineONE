## 💌 About
 
**For My Valentine** is a personal, interactive Valentine's Day website — a creative alternative to traditional gifts. It walks your special person through a charming, multi-step experience: a playful *"Will you be mine?"* proposal with a twist, a reveal of surprise gifts, and a closing video message straight from the heart.
 
Because sometimes the most romantic thing you can do is build something just for them. 💻❤️
 
---
 
## ✨ Features
 
| Feature | Description |
|---|---|
| 🎵 **Ambient Music** | Background music plays automatically to set the romantic mood |
| 💬 **Interactive Proposal** | A playful *"Will you be mine?"* flow — the "No" button doesn't make it easy to say no 😄 |
| 🎁 **Gift Reveal Section** | A curated reveal of three personal gifts, unveiled one by one |
| 🎥 **Video Message** | A closing personal video — the final, most heartfelt surprise |
| ✦ **Smooth Transitions** | Elegant screen-by-screen animations throughout the experience |
| 📱 **Fully Responsive** | Looks beautiful on mobile and desktop alike |
| 🎀 **Custom Theming** | Soft Valentine aesthetic with warm tones and romantic typography |
 
---
 
## 🎭 How It Works
 
The experience unfolds in stages:
 
```
1. Landing Screen     →  "Normal gifts are boring, so I built this instead"
2. Proposal Screen    →  "Will you be mine? 🥺"  (Yes / No buttons)
3. Persistence Screen →  The "No" path gently insists... 😭
4. Final Ask Screen   →  One last chance — "You're everything to me."
5. Valentine Reveal   →  "Happy Valentine's Day, Baby! ♡"
6. Gift Section       →  Three personal gifts revealed
7. Video Finale       →  "You are my forever" — a personal video closes it all
```

## 🚀 Getting Started
 
Want to make your own version for someone special? Here's how:
 
### 1. Clone the Repository
 
```bash
git clone https://github.com/Paim41/valentine.git
cd valentine
```
 
### 2. Personalize It
 
| File | What to Change |
|---|---|
| `index.html` | Names, messages, gift descriptions |
| `love.js` | Flow logic, button behavior |
| `style.css` | Colors, fonts, animations |
| `music.mp3` | Replace with your song |
| `assets/Valentine.mp4` | Replace with your personal video |
 
### 3. Deploy for Free
 
Push to GitHub, then deploy on [Vercel](https://vercel.com) in one click:
 
```bash
git add .
git commit -m "💕 my valentine website is ready"
git push origin main
```
 
Then connect your repo to Vercel — it deploys automatically. Share the link. Watch their reaction. 🥺
